## Setting Armbian  
Setting WiFi, jam dll  
Login Armbian -> input command
```yaml
armbian-config
```

## Maximize penggunaan SDcard atau Flashdisk  
Perinatah ini bertujuan untuk menggunakan seluruh kapasitas yang tersedia sebagai ROOT  
Login Armbian -> input command
```yaml
armbian-tf
```

## Docker
Install Docker   
Login Armbian -> input command
```yaml
armbian-docker
```
